232/** \brief Set the font foreground color.
233 \ingroup bfont
234
235 This function selects the foreground color to draw when a pixel is opaque in

237 format that you intend to use for the image produced.
238
239 \param c The color to use.
240 \return The old foreground color.
241*/
242uint32 bfont_set_foreground_color(uint32 c);

244/** \brief Set the font background color.
245 \ingroup bfont
246
247 This function selects the background color to draw when a pixel is drawn in
248 the font. This color is only used for pixels not covered by the font when
249 you have selected to have the font be opaque.
250
251 \param c The color to use.
252 \return The old background color.
253*/
254uint32 bfont_set_background_color(uint32 c);

256/** \brief Set the font to draw 32-bit color.
257 \ingroup bfont
258
259 This function changes whether the font draws colors as 32-bit or 16-bit. The

261
262 \param on Set to 0 to use 16-bit color, 32-bit otherwise.

328/** \brief Draw a single character of any sort to the buffer.
329 \ingroup bfont
330
331 This function draws a single character in the set encoding to the given

333 exposing the colors and bitdepths desired. This will allow the writing

335

337 \param bufwidth The width of the buffer in pixels.

339 \param bg The background color to use.

341 \param opaque If non-zero, overwrite background areas with black,
342 otherwise do not change them from what they are.
343 \param c The character to draw.
344 \param wide Draw a wide character.
345 \param iskana Draw a half-width kana character.
346 \return Amount of width covered in bytes.
347*/
